# TFK — The Fragrance Kitchen

Reference documentation for the **TFK Operations, Sales & Brand Performance** project.

TFK (The Fragrance Kitchen) is a luxury fragrance house from Kuwait, founded by Sheikh Majed Al-Sabah and part of the Development Holding Company. The brand blends Middle Eastern perfume heritage with French perfumery craft, sold via flagship stores in Kuwait/UAE and online at [tfk.com.kw](https://www.tfk.com.kw) (Shopify).

## Project scope

1. **Sales performance** — revenue, orders, AOV, top products, customer mix via ShopifyQL
2. **Operations** — inventory health, stockouts/negative stock, bundle sync, SKU hygiene
3. **Brand & social** — referral-source performance, social-attributed revenue, brand consistency

## Conventions

- All figures in KWD (3 decimals), Kuwait time (UTC+3)
- ⚠️ Collection handle mismatch: *Signature Collection* = handle `exclusive-collection`; *Exclusive Collection* = handle `secret-collection` — always verify before linking
